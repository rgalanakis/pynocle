Measures the coupling between modules.
Afferent coupling (Ca) is the number of modules that use a given module. 0 can indicate dead code.
Efferent coupling (Ce)is the number of modules a given module imports. A high value can indicate abrittle module.
Instability (I) is the ratio of Ce to total coupling. I = Ce / (Ce + Ca). Indicates themodule's resilience to change between 0 (completely stable) and 1 (unstable).
